Поле   | Обов'язкове |    Тип данних  | WEB|   Опис |

**CODE_CONSTRUCT** | √ | INTEGER       | √ | Код елемента.
**NAME_CONSTRUCT** |   | VARCHAR(96)   | √ | Найменування елемента.
**CODE_PARENT**    |   | INTEGER       |   | Неважливо. ~~Уточнити у СОФТПРОЕКТ~~.
